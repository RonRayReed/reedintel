How It Works in Production

1
Signal ingested — ProZorro or other source returns raw Ukrainian text (title, description, city).
2
Translation triggered — the worker calls the self-hosted LibreTranslate container (already on the same VM) with uk → en.
3
Stored bilingually — both the original Ukrainian and the English translation are saved in the database, preserving fidelity.
4
Dashboard displays English — analysts read clean English signals; original text available on demand for verification.
